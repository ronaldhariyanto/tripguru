# tripguru
TripGuru Frontend Test Challenge

## How to use

### Setup

Install it:

```bash
npm install
```

### Run Development

Run it:

```bash
npm run dev
```

### Open in browser:

```bash
http://localhost:3000/
```

### Change the city
We can change the city with modified the url:

```bash
http://localhost:3000/city/bali
http://localhost:3000/city/phuket
```
